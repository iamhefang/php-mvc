# php-mvc

[帮助文档](https://github.com/iamhefang/php-mvc/wiki)

## 目录结构应为

- 项目根目录
    - libraries
        - php-helpers-VERSION.phar
        - php-mvc-VERSION.phar
    - themes
        - theme1
        - theme2
    - application
        - 项目主包
        - application.config.php
        - xxx.config.php
    - index.php
